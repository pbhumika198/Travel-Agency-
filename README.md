Travel Agency Website

A modern and responsive Travel Agency Website built using HTML, CSS, and JavaScript. This project showcases beautiful destinations, travel packages, and a user-friendly interface for booking trips.

Features
Responsive Design (Mobile + Desktop)
Beautiful Destination Sections
Travel Packages Display
Contact Form
Navigation Bar with Smooth Scrolling
Clean and Modern UI

Technologies Used
HTML5
CSS3 (Flexbox & Grid)
JavaScript (Basic Interactions)
